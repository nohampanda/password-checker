# Password Checker 

## Description
A Python tool that analyzes the strength of a password 
and gives tips to improve it.

## How to use
1. Run `password_checker.py` with Python
2. Enter a password when prompted
3. The program displays a score and improvement tips

## Features
- Checks length (minimum 8 characters)
- Checks for at least one uppercase letter
- Checks for at least one digit
- Checks for at least one symbol (!@#$%...)
- Displays a score out of 4 and a security level

## Technologies
- Python 3
